# My Compose Dictionary (Pathway 2)
Апликација изработена со Jetpack Compose која вклучува:
- **Pathway 1:** State management, Button, Text.
- **Pathway 2:** LazyColumn, Animations (animateContentSize), Material 3 Theming.
- **Функционалност:** Пребарување, преведување и трајно зачувување преку SharedPreferences.
